O mně

Jmenuji se Martin Mrštík, je mi 27 let. Doposud jsem pracoval jako správce informačních technologií ve společnosti zabývající se kovovýrobou.

Ve volném čase se věnuji programování, nyní bych rád z koníčku udělal zdroj obživy. Původně jsem se věnoval jazyku Python, nicméně touha po tom, aby moje práce „byla vidět“, mě přivedla k webovým technologiím.

Nyní se intenzivně vzdělávám v HTML5, CSS3 a moderním JavaScriptu.

Zkušenosti

Většinu zkušeností, stejně tak seznamování s novými technologiemi, získávám z webových kurzů na learn2code.cz, kde jsem si zakoupil doživotní předplatné, a tudíž stálý přísun nových vědomostí.

HTML5, CSS3

Stránky píšu s ohledem na mobile-first a responsivitu. K tvorbě responsivního layoutu využívám kombinaci CSS Grid, Flexbox, případně Bootstrap. K psaní CSS využívám převážně Sass preprocesor.

JavaScript

Nyní se snažím rozvíjet hlavně v moderním JavaScriptu, ale samozřejmě jsem prošel i starší technologie, jako například jQuery, z novějších pak například React.js.
